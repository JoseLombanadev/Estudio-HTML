Creando mi propia reseta con huevos

Esta reseta es muy deliciosa y tiene buenas calorias y carboidratos.

Huevos revuelstos con tomate

Ingredientes:

  • 2 huevos
  • 1 tomate pequeño picado
  • Sal al gusto
  • Un poco de aceite

Preparación:

  1. Bate los huevos con sal.
  2. Calienta el aceite en una sartén.
  3. Agrega el tomate y sofríe por 1 minuto.
  4. Vierte los huevos y revuelve hasta que estén cocidos.

Que es el HTML

HTML es el lenguaje de marcado que estructura el contenido de las páginas web: define títulos, párrafos, imágenes, enlaces y más, para que el navegador sepa qué mostrar y cómo organizarlo.

ventajas del HTML

  • Fácil de aprender y usar
  • Funciona en todos los navegadores
  • No necesita software especial
  • Rápido de cargar y liviano

Mi opinión sobre HTML

Mi opinión real de HTML es que es un lenguaje increible y facil de entender me gusta su estructura, las cosas increibles que puedo aprender y crear con solo etiquetas.
