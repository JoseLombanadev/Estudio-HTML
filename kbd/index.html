Ejercicios para almacenar "kbd"

Ejercicio Uno: Atajos de teclado básicos

  • Ctrl + C
  • Ctrl + V
  • Ctrl + Z
  • Ctrl + S
  • Alt + Tab

Ejercicio Dos: Comandos en consola

Guía pequeña de comandos para que un usuario escriba en la terminal.

  • Inicializar un repositorio

    git init

  • Instalar un paquete

    npm install

  • Limpiar la pantalla

    cls o clear

Ejercicio Tres: Instrucciones paso a paso

Abrir una nueva pestaña y escribir una dirección

Para abrir una nueva pestaña en tu navegador, presiona Ctrl + T (o Command + T si usas Mac). Verás que aparece una pestaña vacía en la parte superior. Luego, haz clic en la barra de direcciones (en la parte superior del navegador), escribe la dirección del sitio web que quieres visitar, por ejemplo www.ejemplo.com, y presiona Enter. ¡Listo! El navegador cargará la página que solicitaste.


Ejercicio Cuatro: Menú interactivo

Menú de atajos, para un editor de texto

  • Abrir la paleta de comandos

    Ctrl + Shift + P

  • Buscar archivo

    Ctrl + p

  • Comentar línea

    Ctrl + /

Ejercicio Cinco: Guía de juego

  • Moverse W, A, S, D
  • Saltar Espacio
  • Disparar Click o Ctrl
